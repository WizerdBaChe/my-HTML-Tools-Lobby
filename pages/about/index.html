關於這個書架

設計理念、使用方式與維護指引

這是什麼

HTML Tools Lobby 是一個以 GitHub Pages 為基礎的靜態工具集合站。 目標很簡單：把散落各處的獨立小工具整理到同一個地方， 讓人可以快速找到並直接使用，不需要安裝任何東西。

大廳頁讀取 tools.json 自動產生工具卡片，完全靜態，不依賴後端。

設計原則

⚡

靜態優先

優先使用靜態 HTML / CSS / JavaScript，不引入不必要的框架，部署即用。

🧩

工具彼此獨立

每個工具位於 tools/<tool-id>/index.html，各自管理資源。外部工具可直接用完整 URL 掛載，不需進 repo。

📋

單一資料來源

tools.json 是工具卡片、精選區塊、排序與狀態的唯一資料來源。新增或調整工具只需更新這個檔案。

🚀

MVP 優先

先做最小可用版，確認不破壞既有結構後，再考慮功能擴充。

新增工具

  • 建立 tools/<tool-id>/index.html
  • 在 tools.json 新增一筆工具資料
  • 外部工具（其他 repo）：path 填完整 URL，supportsIframe 設為 false
  • 從大廳確認卡片顯示、搜尋與分類正常

工具狀態

  • ready功能穩定，可正常使用
  • beta基本功能可用，持續測試中
  • draft開發中，尚未正式發布

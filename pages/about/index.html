關於本站 (About)

這是一個以 GitHub Pages 為基礎的靜態 HTML 工具集合站，用來集中整理多個獨立小工具。

首頁作為 Lobby，負責讀取 tools.json，並提供精選工具、搜尋、分類、排序與 iframe 預覽功能。

本站目標

  • 把零散的小工具集中到同一個 repo 管理。
  • 讓每個工具保有獨立頁面與清楚路徑。
  • 用一致的卡片式入口降低維護成本。
  • 維持靜態站架構，避免不必要的複雜化。

目前功能

Tool Cards

首頁自動根據 tools.json 產生工具卡片。

Featured

可指定精選工具顯示在首頁上方。

Filter & Sort

支援搜尋、分類篩選與排序。

Status

支援 READY、BETA、DRAFT、NEW 等標記。

使用方式

  • 從首頁瀏覽精選工具與全部工具。
  • 使用搜尋框、分類按鈕與排序快速定位工具。
  • 點擊「開啟」進入獨立工具頁。
  • 若工具支援，可直接在首頁預覽。

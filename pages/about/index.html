關於本站 (About)

這是一個以 GitHub Pages 為基礎的靜態 HTML 工具集合站，用來集中整理與展示多個獨立的小工具。

首頁作為 Lobby（大廳頁），負責列出工具卡片、提供快速開啟入口，並讓部分工具可以透過 iframe 方式在首頁內預覽。

本站目標

  • 把零散的小工具集中到同一個 repo 管理。
  • 讓每個工具保有獨立頁面與清楚路徑。
  • 用一致的卡片式入口降低維護成本。
  • 先以 MVP 為主，再逐步增加搜尋、分類、標籤與互動功能。

目前架構

本站採用純靜態架構，主要由 index.html、tools.json、各工具資料夾與基礎頁面組成。

Lobby

首頁負責讀取 tools.json 並產生卡片。

Tools

每個工具獨立放在 tools/<tool-id>/。

Preview

支援的工具可在首頁以 iframe 預覽。

Docs

README 與維護文件負責記錄用途與規則。

使用方式

  • 從首頁瀏覽工具卡片。
  • 點擊「開啟」進入獨立工具頁。
  • 若工具支援預覽，可在首頁小視窗中直接載入。
  • 若連結失效，請返回首頁重新選取工具。

後續方向

  • 工具分類與標籤篩選。
  • 搜尋與排序。
  • 最近新增工具區塊。
  • 更完整的 changelog 與說明文件。
